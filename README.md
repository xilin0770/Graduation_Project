# edu_system (掌柜智库)

基于 LangGraph 和多模态大模型的智能知识库管理与查询系统

## 项目简介

Shopkeeper Brain 是一个为店铺提供智能知识库服务的系统，支持多格式文档导入、知识图谱构建和智能问答。系统采用混合检索策略，融合向量检索、知识图谱查询和网络搜索，为用户提供精准的知识查询服务。

## 功能特性

### 文档导入
- 支持 PDF、Markdown、图片等多种格式
- 复杂 PDF 智能转换（支持表格、公式、图片）
- 自动提取商品名称
- 知识图谱自动构建

### 智能查询
- 多路混合检索（向量检索 + HyDE + 知识图谱 + 网络搜索）
- RRF 融合排序 + 重排序优化
- SSE 流式响应，实时返回答案
- 完整的会话历史管理

## 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                        FastAPI API 层                        │
│            (导入服务 :8000 / 查询服务 :8001)                  │
├─────────────────────────────────────────────────────────────┤
│                      LangGraph 工作流引擎                     │
│          ┌──────────────────┬──────────────────┐            │
│          │    导入流程 (7节点)  │    查询流程 (9节点)  │            │
│          └──────────────────┴──────────────────┘            │
├─────────────────────────────────────────────────────────────┤
│                         工具层                               │
│  ┌─────────┬─────────┬─────────┬─────────┬─────────┐       │
│  │ LLM工具  │ 向量化工具 │ 重排序工具 │ SSE工具  │ 任务管理  │       │
│  └─────────┴─────────┴─────────┴─────────┴─────────┘       │
├─────────────────────────────────────────────────────────────┤
│                        存储层                                │
│  ┌─────────┬─────────┬─────────┬─────────┐                 │
│  │ Milvus  │  Neo4j  │ MongoDB │  MinIO  │                 │
│  │ (向量库) │ (图数据库) │ (文档库) │ (对象存储) │                 │
│  └─────────┴─────────┴─────────┴─────────┘                 │
└─────────────────────────────────────────────────────────────┘
```

## 技术栈

| 类别 | 技术 |
|------|------|
| 后端框架 | FastAPI |
| 工作流引擎 | LangGraph |
| 向量数据库 | Milvus |
| 图数据库 | Neo4j |
| 文档数据库 | MongoDB |
| 对象存储 | MinIO |
| 大模型 | DashScope API (通义千问) |
| 向量化 | BGE-M3 (稠密+稀疏向量) |
| PDF处理 | MiniRu |
